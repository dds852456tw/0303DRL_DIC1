# Wu Yiting's Personal Page - Chatfolio Edition

I have created a premium, interactive personal page for **Wu Yiting**. The design is inspired by the Chatfolio template, featuring a conversational interface that makes exploring your profile an engaging experience.

## ✨ Key Features
- **Conversational UI**: Information is revealed through a simulated chat, complete with typing animations and progressive discovery.
- **Glassmorphism Design**: A modern, translucent interface with soft gradients, backdrop blurs, and premium shadows.
- **Custom 3D Avatar**: A high-resolution, professional AI-generated avatar to represent Wu Yiting.
- **Interactive Time Tracking**: A live clock in the header and dynamic timestamps for every message.
- **Micro-interactions**: Smooth transitions, hover effects on chips, and slide-up message animations.

## 🛠️ Implementation Details
1. **HTML5 Structure**: A clean, semantic layout centered around a `chat-container`.
2. **Vanilla CSS**: Leverages modern CSS features like `backdrop-filter`, `flexbox`, and `keyframes` to achieve a high-end look without external frameworks.
3. **Dynamic Logic (`index.js`)**:
   - Simulates "typing" delays for a more human feel.
   - Automatically scrolls the chat to the latest message.
   - Updates the system time in real-time.
