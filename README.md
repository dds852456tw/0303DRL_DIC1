# 🎨 YiTingWu: Creative Personal Page

A premium, modern, and professional personal page for **YiTingWu**. Inspired by contemporary digital agency aesthetics, this project features a bold "Corporate Creative" style with an organic, vibrant theme.

## 🌟 Preview

The design is based on the [Freepik Digital Marketing Agency Template](https://www.freepik.com/free-psd/digital-marketing-agency-corporate-facebook-instagram-story-template_417546609.htm), adapted for a personal portfolio.

- **Theme**: Sophisticated Olive Green (`#525B44`) paired with Vibrant Lime (`#D4E157`).
- **Typography**: Bold, high-impact headers using **Inter** and **Poppins**.
- **Visuals**: A high-resolution portrait integrated with organic background shapes and glassmorphism elements.

## ✨ Key Features

- **🖼️ Premium Aesthetic**: 
  - Thick rounded frame for a modern "contained" feel.
  - Animated organic "blob" background for dynamic energy.
  - Glassmorphism info cards with backdrop blur.
- **🕒 Real-Time Clock**: A live, auto-updating 24-hour clock in the footer.
- **📱 Fully Responsive**: Seamless transition from desktop to mobile layouts.
- **🚀 Performance-First**: Built with lightweight Vanilla HTML/CSS/JS (no heavy frameworks).

## 🛠️ Technology Stack

- **HTML5**: Structured for clarity and accessibility.
- **CSS3 (Modern)**: CSS Grid, Flexbox, Keyframes, and variable-based design system.
- **JavaScript (ES6)**: Real-time time tracking and smooth entrance animations.
